Boolean Values

  • Empty Class Definition (returns nil)
  • Nonempty Class Definition
  • Method definitions (:m) -NOT result from calling a method
  • Strings
  • nil
  • true (the object)

Empty Class Definition (returns nil)

Evaluates to: False

Nonempty Class Definition

shopping_list = {'fish' => 'Tilapia',

'milk' => '2%'

'sweets' => 'chocolate'

}

Evaluates to: Same value as last value they contain.

Method definitions (:m) -NOT result from calling a method

Evaluates to: True

String boolean value

Evaluates to: True

nil boolean value

Evaluates to: False

true (the object)

Evaluates to: True
